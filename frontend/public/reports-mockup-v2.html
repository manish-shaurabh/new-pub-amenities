Drill-down chain (modals stack)

SuperAdmin → DRO → Supervisor → Station

1. Click DRO ELECTRICAL → opens drawer with that RO's 3 supervisor mini-cards (Section 2 style).
2. Click a supervisor card → opens new drawer (or expands within) showing 3 station cards (Section 1 style).
3. Click a station's location bar → drills down to the asset list filtered to that location.
4. Each level has its own "Download PDF / Excel" button — exports the data visible at that level.

📋 Confirmed design — what I'll build

  • ✅ Center % = (working + yellow) / total; smooth gradient color from deep red (≤80%) → green (100%)
  • ✅ SUP card: concentric rings per asset type; W shown as light grey + Y/O/R emphasized
  • ✅ Admin/SA card: concentric rings per department (only ONE ring if RO covers single dept, multiple if cross-dept)
  • ✅ Below donut: horizontal stacked bars per location, ordered worst-first
  • ✅ RO grid: mini-summary card per supervisor → click → drawer opens with full station cards
  • ✅ Admin/SA grid: mini-summary card per RO → click → drawer with supervisor cards → click → station cards
  • ✅ Stations with 0 assets: hidden entirely
  • ✅ Stations with 0 defects: "✓ ALL CLEAR" badge instead of empty rings
  • ✅ PDF export: cover summary + one page per station; Excel: drilled to per-asset rows
  • ✅ Server-side: ReportLab for PDF, openpyxl for Excel
